11.3.131 nonvar/1   [ISO]

Synopsis

nonvar(+Term)

Term is currently instantiated.

Arguments

Term
term

Examples

     | ?- nonvar(foo(X,Y)).
     
     true ;
     no
     | ?- nonvar([X,Y]).
     
     true ;
     no
     | ?- nonvar(X).
     
     no
     | ?- Term = foo(X,Y), nonvar(Term).
     
     true ;
     no

Exceptions

None.

See Also

ref-lte-met-typ.
