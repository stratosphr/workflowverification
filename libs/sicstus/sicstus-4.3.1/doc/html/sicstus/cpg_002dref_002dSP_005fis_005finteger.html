12.3.49 SP_is_integer()

Synopsis

     #include <sicstus/sicstus.h>
     
     int
     SP_is_integer(SP_term_ref term);

Determines whether the representation of term is a Prolog integer.

Arguments

term
The SP_term_ref to be inspected

Return Value

1 if it is a integer and 0 otherwise.

Please note: SP_is_integer() will return true also for integers that are too large to be passed to SP_get_integer(). In this case you will need to use SP_get_integer_bytes() to obtain the representation.
