12.3.13 SP_error_message()

Synopsis

     #include <sicstus/sicstus.h>
     char const *
     SP_error_message(int errnum);

Obtains a pointer to the diagnostic message corresponding to a specified error number.

Arguments

errnum
The error number.

Return Value

A pointer to the diagnostic message.

CPL Notes.
