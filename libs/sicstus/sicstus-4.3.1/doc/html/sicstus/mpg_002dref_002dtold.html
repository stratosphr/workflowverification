11.3.236 told/0

Synopsis

told

Closes the current output stream.

Description

Current output stream is set to be user_output; that is, the user's terminal.

Always succeeds

Exceptions

None.

Examples

See Also

tell/1, close/[1,2], current_output/1, ref-iou-sfh.
