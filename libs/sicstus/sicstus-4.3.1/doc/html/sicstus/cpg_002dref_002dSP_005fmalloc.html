12.3.55 SP_malloc()

Synopsis

     #include <sicstus/sicstus.h>
     
     void *
     SP_malloc(size_t size);

Allocates a block of at least size bytes.

Arguments

size
Requested number of bytes.

Return Value

NULL on failure, the pointer otherwise.

See Also

See OS Memory Management.
