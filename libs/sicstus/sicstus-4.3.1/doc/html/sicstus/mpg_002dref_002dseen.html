11.3.202 seen/0

Synopsis

seen

Closes the current input stream.

Description

Current input stream is set to be user_input; that is, the user's terminal.

Always succeeds

Exceptions

None.

Examples

See Also

see/1, close/[1,2], current_input/1, ref-iou-sfh.
