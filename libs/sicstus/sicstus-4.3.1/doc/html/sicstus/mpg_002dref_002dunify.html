11.3.240 =/2   [ISO]

Synopsis

+Term1 = +Term2

unifies Term1 and Term2.

Arguments

Term1
term
Term2
term

Description

This is defined as if by the clause ‘Z = Z.’.

If =/2 is not able to unify Term1 and Term2, it will simply fail.

Exceptions

None.

See Also

Glossary, ref-sem-occ.
