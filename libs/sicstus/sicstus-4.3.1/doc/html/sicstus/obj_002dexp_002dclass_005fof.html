10.22.6.8 class_of/2

Synopsis

class_of(+Obj, -Class)

Arguments

Obj
object
Class
atom

Description

Class is the class of Obj.

Exceptions

instantiation_error
Obj is unbound.
type_error
Obj is not a valid object.
