12.3.29 SP_get_current_dir()

Synopsis

     #include <sicstus/sicstus.h>
     
     char *
     SP_get_current_dir(void);

Obtains an encoded string containing the absolute, slash (/) terminated, path to the current working directory. The return representation is allocated with SP_malloc and should be freed with SP_free.

Return Value

The string on success and NULL on error.

cpg-ref-SP_set_current_dir. OS File System.
