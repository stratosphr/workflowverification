12.3.7 SP_cons_functor_array()

Synopsis

     #include <sicstus/sicstus.h>
     
     int
     SP_cons_functor_array(SP_term_ref term,
     		      SP_atom name,
     		      int arity,
     		      SP_term_ref *arg);

Assigns to term a reference to a compound term whose arguments are the elements of arg. If arity is 0, assigns the Prolog atom whose canonical representation is name. This is similar to calling =../2 with the first argument unbound and the second argument bound.

Arguments

term
The SP_term_ref to be assigned
name
The name of the functor
arity
The arity of the functor
arg
The argument array

Return Value

Zero if the conversion fails (as far as failure can be detected), and a nonzero representation otherwise.

Creating Prolog Terms.
