4.3.4.7 Module Declarations

One of the following declarations: 

     :- module(+ModuleName, +ExportList).
     
     :- module(+ModuleName, +ExportList, +Options).

where ExportList is a list of predicate specs, declares that the forthcoming predicates should go into the module named ModuleName and that the predicates listed should be exported. See ref-mod for details. See mpg-ref-meta_predicate.
