13.7 splm — SICStus Prolog License Manager

Synopsis

     % splm -i Site
     % splm -a LicensedProduct ExpirationDate Code

Description

SICStus Prolog requires a license code to run. You should have received from SICS your site name, the expiration date and the code. This information is normally entered during installation, but it can also be entered later on by means of this command-line tool.

Under Windows, splm must be run by a user with Administrative rights. The windowed version of SICStus (spwin.exe) has a menu item for license entry, making splm unnecessary under Windows.

Please note: when using spwin.exe for changing the license information, it too must be run with Administrative rights. This is especially important under Windows Vista and later.
