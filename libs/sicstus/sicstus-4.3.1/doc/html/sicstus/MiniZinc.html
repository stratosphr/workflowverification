10.46.3 MiniZinc

  • MiniZinc Prerequisites: Prerequisites
  • MiniZinc Exported Predicates: Exported Predicates

The predicates described here make it possible to load and run MiniZinc programs directly from within SICStus and require an external MiniZinc-to-FlatZinc translator. In this way, the predicates described here are essentially wrappers to the predicates described in FlatZinc, handling the MiniZinc-to-FlatZinc translation and providing a more high-level interface.
