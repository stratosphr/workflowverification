12.3.82 SP_put_term()

Synopsis

     #include <sicstus/sicstus.h>
     
     int
     SP_put_term(SP_term_ref to,
                 SP_term_ref from);

Assigns to to the representation of from.

Arguments

to
The SP_term_ref to be assigned
from
The SP_term_ref whose representation is accessed

Return Value

Zero if the conversion fails (as far as failure can be detected), and a nonzero representation otherwise.

Creating Prolog Terms.
