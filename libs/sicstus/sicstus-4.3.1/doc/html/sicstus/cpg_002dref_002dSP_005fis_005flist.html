12.3.50 SP_is_list()

Synopsis

     #include <sicstus/sicstus.h>
     
     int
     SP_is_list(SP_term_ref term);

Determines whether the representation of term is a Prolog list cell, i.e. a compound term with functor ./2.

Arguments

term
The SP_term_ref to be inspected

Return Value

1 if it is a list and 0 otherwise.

See Also

Testing Prolog Terms.
