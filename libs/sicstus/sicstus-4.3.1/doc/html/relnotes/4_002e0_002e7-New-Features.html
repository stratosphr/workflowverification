13.10.1 New Features

  • Most text streams can now be opened with reposition(true), allowing set_stream_position/2 and seek/4. This works for fixed-width, single-byte encodings. This includes Latin 1 and similar encodings provided LFD is used for end-of-line. See open.
  • library(clpb) is revived from SICStus 3, unsupported.
