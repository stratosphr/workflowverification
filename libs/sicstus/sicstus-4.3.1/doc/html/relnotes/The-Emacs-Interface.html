12 The Emacs Interface

The Emacs Interface was originally developed for GNU Emacs 19.34 and is presently being maintained using GNU Emacs. For best performance and compatibility and to enable all features we recommend that the latest versions of GNU Emacs be used. For information on obtaining GNU Emacs ; see http://www.gnu.org/software/emacs/.
